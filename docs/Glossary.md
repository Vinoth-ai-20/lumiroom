## Terms and Abbreviations

| Term | Definition |
|------|------------|
| **ARCore** | Google's proprietary platform for building augmented reality experiences on Android. |
| **SceneView** | An open-source 3D engine for Android, used by Lumiroom as the primary AR rendering surface, built on top of Google Filament. |
| **FMP** | Furniture Mega Pack. A third-party repository of 3D furniture models. |
| **GLB** | The binary version of the glTF (GL Transmission Format) 3D asset format. Used to store mesh and texture data in a single file. |
| **Draco** | An open-source library created by Google for compressing 3D geometric meshes and point clouds. |
| **KTX2** | A texture format designed for optimal GPU loading. |
| **Room DB** | An abstraction layer over SQLite provided by Android Jetpack. |
| **UDF** | Unidirectional Data Flow. An architectural pattern where state flows down to the UI, and events flow up to the ViewModel. |
| **Hit Test** | The process of projecting a 2D screen coordinate into 3D space to detect intersection with a real-world physical plane (e.g., the floor). |
| **Raycast** | Similar to a Hit Test, firing a virtual line from the camera center into the environment. |
| **ToF Sensor** | Time-of-Flight sensor. Hardware on modern smartphones used to calculate distance and depth accurately. |
